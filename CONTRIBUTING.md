# Contributing to Awesome OpenClaw

Thank you for your interest in contributing! This project aims to be the most comprehensive collection of OpenClaw resources.

## How to Contribute

### Adding Resources

1. **Fork** this repository
2. **Add** your resource to the appropriate section
3. **Submit** a Pull Request

### Quality Guidelines

Resources should be:

- **Relevant** - Directly related to OpenClaw (or Clawdbot/Moltbot)
- **Working** - Links must be functional
- **Valuable** - Provides genuine value to the community
- **Appropriate** - No spam, malware, or inappropriate content

### Format

#### For links in README.md tables:

```markdown
| Title | [Description](https://example.com) | Category |
```

#### For community section files:

```markdown
| Title | Link | Engagement | Notes |
|-------|------|------------|-------|
| Example Post | [Thread](https://...) | 100 upvotes | Brief note |
```

### Categories

Add resources to the most appropriate category:

- **Official Resources** - Official OpenClaw links
- **Documentation** - Docs and references
- **Tutorials & Guides** - How-to content
- **YouTube Videos** - Video tutorials
- **Reddit** - Reddit discussions
- **Articles** - Blog posts and media coverage
- **Skills** - Extensions and plugins
- **Model Providers** - LLM integrations
- **Hosting & Deployment** - Deployment options
- **Tools** - Related tools and integrations
- **Use Cases** - Real-world examples
- **Security** - Security resources

### What We're Looking For

- New tutorials and guides
- International resources (non-English)
- Community tools and integrations
- Security best practices
- Performance optimization tips
- Real-world use case examples
- Video tutorials and demos

### What We Don't Accept

- Duplicate resources
- Broken links
- Promotional/spam content
- Malicious tools or skills
- Low-quality or AI-generated filler content

## Reporting Issues

If you find:

- **Broken links** - Open an issue or PR to fix
- **Outdated information** - Submit an update
- **Malicious content** - Report immediately via issue

## Code of Conduct

Be respectful and constructive. This is a community resource.

## Questions?

Open an issue or join the [OpenClaw Discord](https://discord.com/invite/clawd).
